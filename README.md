# go-cfgreader

Run:

```
$ go get github.com/velp/go-cfgreader
$ cd $GOPATH/src/github.com/velp/go-cfgreader
$ make build
$ ./_output/cfgreader -c test.yaml
Config struct: &config.Config{Username:"test", Password:"test", Url:"test"}
```

Install deps:

```
$ make deps
```

Run test

```
$ make test
```
